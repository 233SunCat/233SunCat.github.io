```go
直接插入排序-使用顺序排序
r = []
voie insertSort(){
	var shaozi int
	for (i=1;i<len(r);i++){
		shaozi = r[i]//待插入元素放入监视哨中
		for (j=i-1;shaozi < r[j];j--){r[j+1] = r[j]}//比shaozi元素大的都后退
		r[j+1] = shaozi
	}
}
算法分析：
最好的比较次数：1次，最坏的比较次数：i-1次     
最好的移动次数：0次，最坏的移动次数：i-1次
时间复杂度O(N^2)
空间复杂度O(1)
```
